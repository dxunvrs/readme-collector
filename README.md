# Описание
Добавление лаб и сборщик файла README.md  
Поддерживает две функции: добавление лабы в таблицу предмета и сборка README.md  
В README.md добавляет все таблицы и обновляет список  
# Требования
Рабочая директория должна содержать папки с названиями предметов, как в Paths.py  
Если в них нет md-таблиц, то программа сама их создаст с пустым шаблоном таблицы  
# Инструкция
1. Добавить свои пути в файле src/Paths.py, по умолчанию выберется домашний каталог  
По необходимости добавить новый предмет в список subjects, но для добавления лаб нужно будет самому дописывать LabGen.py)  

2. Можно запустить файл LabGen.py и добавить лабу в заданную предметную таблицу
```
python3 src/LabGen.py
Название: prog
Номер: 1
Макс.баллов: 100
Мои баллы: 100
Дата сдачи: xx.xx.xxxx
Тема: Указываем тему
```
3. Файлом ReadmeUpdate.py пересобрать README.md
Шаблон для READ.md - src/Outline.md
